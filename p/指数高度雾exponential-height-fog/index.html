指数高度雾（Exponential Height Fog）

缓存的间接光照采样，用于模拟动态对象和未构建场景预览的全局光照效果。

指数高度雾在地图的低处创造更多的密度，在高处创造更少的密度。过渡是平稳的，所以当你增加高度时，你永远不会遇到一个硬的切断。指数高度雾也提供了两种雾色：一种时面向主导方向光的半球（或者如果不存在直接向上），另一种时面向相反半球的颜色。

属性描述
雾密度（Fog Density）此为整体密度稀疏，是可视雾层的厚度。
雾高度衰减（Fog Height Falloff）高度密度系数，控制高度降低时密度增加的成都。值越小，过渡就越大。
高度雾偏移（Height Fog Offset）此控制相对于Actor放置Z（高度）的雾层高度偏移。
